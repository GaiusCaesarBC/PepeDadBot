## 🤖 BOT FEATURES

The ARCHIVE SYSTEM allows investigators to interact directly with the investigation.

### Commands

- `/start` → Initialize system interface  
- `/status` → Current system status  
- `/clue` → Retrieve a random clue  
- `/decode` → Decode Base64 transmissions  
- `/scan` → Scan for anomalies  
- `/rank` → Investigator rankings  
- `/report` → Submit findings
